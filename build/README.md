# Build Resources

이 폴더에는 앱 빌드에 필요한 리소스들이 들어갑니다.

## 필요한 파일들:

### icon.png
- 앱 아이콘 (1024x1024 픽셀)
- macOS: .icns 파일로 자동 변환
- Windows: .ico 파일로 자동 변환

### 아이콘 요구사항:
- 크기: 1024x1024 픽셀
- 형식: PNG (투명 배경)
- 내용: MemoBee 벌 아이콘 🐝
